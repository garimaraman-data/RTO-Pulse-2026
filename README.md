# 📊 RTO Pulse 2026: AI-Driven Housing Market Analysis
**An End-to-End Data Engineering & Analytics Pipeline**

## 🎯 Project Overview
This project quantifies the impact of 2025-2026 Return-to-Office (RTO) mandates on the Seattle and Bellevue housing markets. By processing 30,000+ King County records, I identified the "Policy Premium" using automated data pipelines and AI-generated sentiment analysis.

## 🛠️ Technical Stack
* **Data Warehouse:** Snowflake (Medallion Architecture: Bronze, Silver, Gold)
* **AI/ML:** Snowflake Cortex (LLM-based market summarization)
* **Transformation:** Advanced SQL (Window Functions, CTEs, Data Cleansing)
* **Visualization:** Tableau Desktop (Interactive Executive Dashboard)

## 🧠 Key Features
* **Automated Cleaning:** Resolved a 12-month government data lag through SQL-based market simulation.
* **AI Insights:** Leveraged Llama 3 (via Cortex) to generate qualitative investment summaries for every market phase.
* **Interactive UX:** Tableau dashboard featuring dynamic AI tooltips and "Era-based" filtering.
